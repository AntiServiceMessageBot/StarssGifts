# Инструкция для деплоя на Bothost.ru

## Переменные окружения (Environment Variables)

Добавьте эти переменные в настройках вашего бота на Bothost:

1. **BOT_TOKEN** — токен вашего бота от @BotFather
   - Пример: `8464513522:AAGirtWLQyTNrQqwMrkI2RDai0yPwDpgUQg`

2. **DATABASE_URL** — строка подключения к базе данных
   - Для SQLite: `sqlite+aiosqlite:///./marketplace.db`
   - (для бесплатного тарифа используется SQLite)

3. **WEBAPP_URL** — URL вашего WebApp мини-приложения
   - Пример: `https://your-bot.bothost.ru`
   - URL будет доступен после первого деплоя

4. **ADMIN_ID** — ваш Telegram ID или username
   - Пример: `timyr2011`
   - Используется для доступа к админ-панели

## Настройка на Bothost.ru

1. Войдите в ваш аккаунт на bothost.ru
2. Создайте нового бота
3. Подключите GitHub репозиторий с проектом
4. В настройках переменных окружения добавьте все 4 переменные выше
5. Установите команду запуска: `python run.py`
6. Сохраните и дождитесь деплоя

## После первого деплоя

1. Скопируйте URL вашего WebApp из панели Bothost
2. Обновите переменную WEBAPP_URL в настройках Bothost
3. Перезапустите бота
4. Протестируйте: отправьте `/start` боту в Telegram
